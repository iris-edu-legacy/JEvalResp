JEvalResp Installation

Version 1.80

     JEvalResp is a Java version of the 'evalresp' program and, like the original, it is designed to evaluate the response information in ASCII "RESP" files generated by rdseed (V4.16 and above) or station.xml formatted files.  JEvalResp is also capable of fetching and processing responses from web-services servers.
